## Getting Started

```
git clone https://github.com/jaakkolantero/weather-station.git
cd weather-station
npm install
```

create .env to project root

```
touch .env
```

.env

```
OPENWEATHERMAP_API_KEY_PROD=123456789abcdefghijkl
```

run locally

```
npm run dev
```

[localhost:3000](http://localhost:3000)
